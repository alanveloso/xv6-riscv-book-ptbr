\chapter{Resumo}
\label{CH:SUM}

Este texto apresentou as principais ideias em sistemas operacionais ao estudar um sistema operacional, o xv6, linha por linha. Algumas linhas de código incorporam a essência das principais ideias (por exemplo, troca de contexto, limite usuário/kernel, bloqueios, etc.) e cada linha é importante; outras linhas de código fornecem uma ilustração de como implementar uma ideia específica de sistema operacional e poderiam ser feitas de maneiras diferentes (por exemplo, um algoritmo melhor para escalonamento, melhores estruturas de dados em disco para representar arquivos, melhor registro para permitir transações concorrentes, etc.). Todas as ideias foram ilustradas no contexto de uma interface de chamada de sistema particular e muito bem-sucedida, a interface Unix, mas essas ideias se aplicam ao design de outros sistemas operacionais.
